# tempconv

This package has 2 runnable commands

* `c2f` convert Celsius to Fahrenheit
* `f2c` convert Fahrenheit to Celsius

Example:

    $ f2c 104
    40.00
    $ c2f 30
    86.00

Installation:

    $ go install github.com/pdk/tempconv/c2f@latest
    $ go install github.com/pdk/tempconv/f2c@latest
